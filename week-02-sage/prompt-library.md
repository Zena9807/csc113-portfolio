# My Prompt Library
Last updated: 2026-02-15

## Learning & Studying

### 1. Bestie Tutor
**Prompt:**
Act like my best friend. Use African American Vernacular and explain **[topic]** in plain language, then give a real-world IT/cybersecurity example. End with 2 quick questions to check my understanding.

**Use this when:** I’m learning a new concept and need it explained clearly, but in a friendly way.  
**Works best with:** Any

### 2. Acronym / Mneumonic Device
**Prompt:**
Create an acronym or mneumonic device to help me understand **[insert concept]**. Then ask me to explain it step using the acronym or mneumonic device.

**Use this when:** I'm trying to memorize a process 
**Works best with:** Gemini and Chat GPT

## Creating & Building

### 1. Practice Questions Builder
**Prompt:**
Create 6 practice questions for **[topic]**: 3 multiple choice and 3 scenario-based. Then check my responses and give feedback.

**Use this when:** I’m preparing for a quiz/exam.  
**Works best with:** Any

### 2. Summarized Study Guide
**Prompt:**
Create a summary of all subject matter discussed in preparation for a Quiz on **[topic]**: Then build a fill in the blank version using real world scenarios.

**Use this when:** I’m preparing for a quiz/exam.  
**Works best with:** Any

## Evaluating & Improving

### 1. Review and Check Work
**Prompt:**
Review the following information and check my work: Then note any incorrect answers and give an explaination on why.

**Use this when:** I’m working on efficiency and understanding  
**Works best with:** Chat GPT

### 2. Project Improvement
**Prompt:**
Here is a copy of my current timeline for **[project/study timeline]**: Then show me 5 ways to improve it with ADHD guidelines.

**Use this when:** I'm working on time management for a project or personal project.  
**Works best with:** Chat GPT

## Users Choice

### 1. As You Like It
**Prompt:**
Zenafi it! Give me the full breakdown of **[concept/steps]**. I need step by step instructions how to to do it. Then create a real-world scenario question to complete the task. Time it and tell me how long it took me to complete after sending.

**Use this when:** I want to work on speed and accuracy 
**Works best with:** Gemini

### 2. Best Throw
**Prompt:**
I'm creating a fragrance with **[Fragrance Profile]**. Create a formulation for a 10ml oil roller bottle. Include the amount for each scent, formulation set time and generate for the strongest throw.

**Use this when:** Creating a custom fragrance oil 
**Works best with:** Chat GPT
